Drop Test Analysis and Experimental Validation in Ansys

A comprehensive engineering validation project comparing digital Ansys simulation results with real-world physical drop test data.

The Challenge

The primary objective was to ensure that the digital drop test simulations performed in Ansys accurately mirrored real-world physical impacts. To do this, we needed a reliable way to capture the physical responses of the test object during a live drop and translate those findings into actionable data to compare against the simulation models.

The Solution

To validate the Ansys simulation, we instrumented the physical test subject with a combination of sensors. We captured precise strain values using a strain gauge aligned to specific axes. Additionally, we utilized an accelerometer and a gyroscope to track the exact motion, impact forces, and rotational dynamics. By comparing this real-life sensor data directly with the theoretical simulation outputs, we successfully validated the accuracy of the Ansys drop test models.
